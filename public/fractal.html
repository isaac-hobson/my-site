> FRACTAL GENERATOR INFO

> WHAT ARE FRACTALS?

Fractals are infinitely complex patterns that are self-similar across different scales. They are created by repeating a simple process over and over in an ongoing feedback loop.

> HOW TO USE

1. Select a preset from the dropdown, or write custom code

2. Click GENERATE to render the fractal

3. Use AUTO ZOOM to infinitely zoom into detailed areas

4. Click and drag to pan, scroll to zoom manually

> CUSTOM CODE SYNTAX

The generator uses GLSL (OpenGL Shading Language). Available variables:

z    - Current complex number (vec2)
c    - Constant/seed point (vec2)  
i    - Current iteration (int)

// Complex operations:
z.x  - Real part
z.y  - Imaginary part

// Example formulas:
// Mandelbrot: z = z^2 + c
z = vec2(z.x*z.x - z.y*z.y, 2.0*z.x*z.y) + c;

// Julia: (use preset - Julia requires special init)
// z starts at pixel position, c is constant
z = vec2(z.x*z.x - z.y*z.y, 2.0*z.x*z.y) + c;

// Burning Ship: z = (|Re(z)| + i|Im(z)|)^2 + c
z = vec2(abs(z.x), abs(z.y));
z = vec2(z.x*z.x - z.y*z.y, 2.0*z.x*z.y) + c;

// Tricorn: z = conj(z)^2 + c
z = vec2(z.x*z.x - z.y*z.y, -2.0*z.x*z.y) + c;

> PRESET DETAILS

Mandelbrot Set: The classic fractal. Points where z = z^2 + c remains bounded.

Julia Set: Related to Mandelbrot but uses a fixed constant. Creates varied organic shapes.

Burning Ship: Uses absolute values, creating flame-like asymmetric patterns.

Tricorn: Uses complex conjugate, producing three-cornered bulbs.
